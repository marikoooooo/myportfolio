The Brant Hub

A comprehensive redesign of BScene's community event guide website, enhancing user experience for the Brantford, Brant, and Six Nations community

Figma Adobe InDesign UI/UX Design Prototyping

Project Overview

BScene is "the #1 EVENT GUIDE for BRANTFORD | BRANT | SIX NATIONS," serving as a crucial community resource for local events, stories, and business promotion. The original website at bscene.ca needed a modern redesign to better serve the community and improve user experience for finding local events and information.

My role in this client project was to create a comprehensive Figma prototype that would modernize the site's interface while maintaining its essential function as a community hub. The redesign focused on improving navigation, enhancing event discovery, and creating a more engaging visual experience that reflects the vibrant local community it serves.

Design Time

4-5 weeks

Key Design Improvements

Enhanced Event Discovery

Redesigned the event browsing experience to make it easier for community members to find local events, with improved filtering and search capabilities.

Community-Focused Design

Created a visual identity that celebrates the local Brantford, Brant, and Six Nations community while maintaining professional credibility.

Streamlined Information Architecture

Reorganized content to better highlight events, stories, and local business features, making the site more effective as a community resource.

Design Process

The redesign process began with analyzing the existing BScene website and understanding the community's needs for finding local events and information. I studied user behavior patterns, examined the current site structure, and identified key pain points in the event discovery process.

Tools & Methods

  • Design: Figma for wireframing, prototyping, and visual design
  • Layout: Adobe InDesign for content organization and print materials
  • Research: User interviews, usability testing, competitive analysis
  • Methodology: Design thinking, user-centered design principles

Design Challenges & Solutions

The main challenge was balancing the need for modern design with the site's established role as a trusted community resource. I solved this by maintaining familiar content organization while introducing cleaner visual design, improved navigation, and better event categorization to enhance the user experience without alienating existing users.

Design Principles

Key Design Principles Applied:
• Clarity: Clear visual hierarchy and content organization
• Consistency: Unified design language across all pages
• Accessibility: WCAG compliant color contrast and typography
• Usability: Intuitive navigation and user flow
• Brand Alignment: Visual design reflecting organization values

What I Learned

Client Collaboration

Prototype Development

Future Improvements

If I were to continue developing this BScene redesign, I would focus on:

  • Event calendar integration with Google Calendar and other popular calendar apps
  • Community member profiles and personalized event recommendations
  • Enhanced local business directory with improved search and filtering
  • Social sharing features to help events reach broader community audiences

  • Integration with social media platforms for event promotion
