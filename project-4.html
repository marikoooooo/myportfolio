Personal Portfolio Website

A responsive, multi-page portfolio website showcasing my work and skills

HTML CSS JavaScript Cursor Git

Project Overview

This personal portfolio website serves as a comprehensive showcase of my skills, projects, and professional journey. Built from scratch using modern web technologies, it demonstrates my ability to create responsive, user-friendly websites with clean design and smooth functionality.

The website features a multi-page architecture with dedicated sections for projects, resume, contact information, and my creative work through Daedalus Studios. It's designed to provide visitors with an intuitive way to explore my work and get in touch with me.

Development Time

2-3 weeks

Team Size

Solo Project

Project Type

Personal Project

Key Features

Responsive Design

Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices.

Smooth Animations

Thoughtful CSS animations and transitions that enhance user experience without being distracting.

Project Showcase

Dedicated project pages with detailed descriptions, technologies used, and links to live demos.

Professional Navigation

Intuitive navigation system with mobile-friendly hamburger menu for easy browsing.

Technical Implementation

The website was built using semantic HTML5, modern CSS3 features, and vanilla JavaScript. I focused on creating clean, maintainable code while ensuring optimal performance and accessibility.

Technologies Used

  • Frontend: HTML5, CSS3, JavaScript ES6+
  • Styling: CSS Grid, Flexbox, Custom Properties
  • Animations: CSS Transitions and Keyframes
  • Development: Cursor AI-powered code editor
  • Version Control: Git & GitHub
  • Fonts: Google Fonts (EB Garamond)

Development Process

I followed a mobile-first approach, starting with the mobile design and progressively enhancing for larger screens. The development process included wireframing, prototyping, and iterative testing to ensure a polished final product.

Code Highlight

/* CSS Grid for responsive project layout */
.projects-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 2rem;
    margin-top: 2rem;
}

/* Smooth fade-in animations */
.fade-in {
    opacity: 0;
    transform: translateY(20px);
    transition: opacity 0.6s ease, transform 0.6s ease;
}

.fade-in.visible {
    opacity: 1;
    transform: translateY(0);
}
